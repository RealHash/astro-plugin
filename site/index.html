Astro Symbols Add-on for Google Docs

Astrological symbols inside Google Docs, with a visual sidebar that actually feels like the product.

Astro Symbols opens a rich sidebar inside Google Docs so users can browse zodiac signs, planets, aspects, and related glyphs, then insert them into the current document in one click.

Product summary

Core workflow

The add-on opens from Google Docs, renders a visual sidebar, and inserts a selected symbol or notation into the active document.

Stored preferences

Per-user settings are stored through Google Apps Script PropertiesService, including favorites and recent symbols.

Supported languages

The current interface is prepared for Spanish and English users.

Requested permissions

  • script.container.ui Needed to show menus and the sidebar inside Google Docs.
  • documents.currentonly Limited access to the open document so the add-on can insert content where the user asks.

What the app does not do

  • No Drive-wide scope The app does not request general Drive access or Gmail access.
  • No data sale The service is not designed to sell personal data or build ad-targeting profiles.
  • No public checkout today There is currently no public in-app billing or subscription workflow enabled on this add-on.

Billing and commercial status

Astro Symbols is currently presented as an early-stage Google Docs add-on. At the time of this publication, there is no public checkout flow, recurring subscription, or per-use charge exposed through the add-on.

If a paid plan is introduced later, this site and the corresponding Google listing will be updated before any purchase, renewal, or cancellation workflow is enabled.
